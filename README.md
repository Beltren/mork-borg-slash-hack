# MORK BORG /hack Combat

A small Foundry VTT module for using Walton Wood's [`/hack`](https://urizenic.itch.io/slash-hack) combat procedure with the `morkborg` system.

This module is a digital aid only. Read and use the original `/hack` material for the rules text:

https://urizenic.itch.io/slash-hack

## Usage

1. Enable the module in a MORK BORG world.
2. Open an actor sheet.
3. Click a weapon **Attack** button or the **Defend** button.
4. Fill in the `/hack` dialog.
5. Roll and resolve the chat result at the table.

The module does not automatically change HP, degrade armor, break weapons, or apply other lasting consequences.

Dialog values are remembered per browser session, player, actor sheet, and action.
